REA : Regional atmospheric environment (Beta)

About

REPT is a Python framework, built around the WRF-CMAQ, to produce gliding specific weather/air quality forecasts. It automates all the steps involved; downloading the GFS input data, running WRF-CMAQ, post-processing the model output, and creating maps/time series/soundings. REPT is still in development: we are busy testing the code, validating the setup, etc. Anyone who is interested in using REPT is welcome to do so - we would be happy to help - but don't expect a finished product :-) When the time is right, an official 1.0 version will be released.
