Regional atmospheric environment (Beta)

About

REPS is a Python framework, built around the WRF-CHEM/CMAQ/CHIMERE, to produce gliding specific weather/air quality forecasts. It automates all the steps involved; downloading the GFS input data, running AQM, post-processing the model output, and creating maps/time series/soundings. REPS is still in development: we are busy testing the code, validating the setup, etc. Anyone who is interested in using REPS is welcome to do so - we would be happy to help - but don't expect a finished product :-) When the time is right, an official version will be released.
